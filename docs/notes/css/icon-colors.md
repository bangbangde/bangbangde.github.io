# 小图标变色方案

日常使用图标的常见方式有：

- `<img />` 标签
- 使用 `background-image`
- 使用图标字体
- 使用 `SVG` 标签

图标字体不谈，直接设置 `color` 就行了。img 标签和 background-image 本质上是一样的，所以主要是研究：
1. img 元素变色
2. SVG 元素变色
